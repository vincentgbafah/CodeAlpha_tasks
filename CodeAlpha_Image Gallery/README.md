# Image Gallery

A responsive image gallery built with **HTML**, **CSS**, and **JavaScript**.

## Features

- Responsive gallery layout using CSS Grid
- Hover effects with smooth transitions
- Category filters (`All`, `Nature`, `City`)
- Lightbox preview for selected images
- Next/Previous navigation in lightbox
- Keyboard support in lightbox:
  - `Esc` to close
  - `←` / `→` to navigate

## Project Structure

- `index.html` – page structure and gallery markup
- `styles.css` – layout, responsive styles, transitions, and lightbox styles
- `script.js` – filtering, lightbox open/close, and image navigation logic

## How to Run

1. Open the project folder in your browser or editor.
2. Double-click `index.html` to run it directly in a browser.

No build tools or dependencies are required.

## Notes

- Images are loaded from [Picsum](https://picsum.photos/).
- The gallery uses lazy loading for better performance.
