# Portfolio Webpage

A simple personal portfolio webpage built with HTML and CSS.

## Project Structure

- `index.html` — Main webpage markup
- `styles.css` — Styling for the webpage

## Getting Started

1. Open the project folder.
2. Open `index.html` in your browser.

## Edit the Content

- Update text and sections in `index.html`.
- Adjust colors, spacing, and layout in `styles.css`.

## Deployment

You can deploy this static site on platforms like:

- GitHub Pages
- Netlify
- Vercel
